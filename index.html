Exemplos de Flexbox

Explore as principais propriedades do Flexbox aplicadas a um container com quatro imagens.

display: flex

Define o container como um flex container.

flex-direction: column

Coloca os itens em uma coluna (uma imagem embaixo da outra).

flex-direction: row-reverse

Inverte a ordem dos itens horizontalmente.

justify-content: flex-end

align-items: flex-end
