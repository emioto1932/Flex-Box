Exemplos de Propriedades Flexbox

display: flex

Define o container como um flex container.

Define se os itens devem quebrar linha.

flex-direction

Define a direção dos itens no container. Respectivamente:

flex-direction: row; flex-direction: column;

justify-content

Alinhamento horizontal dos itens. Respectivamente:

justify-content: flex-start; justify-content: flex-end; justify-content: center; justify-content: space-between;

align-items

Alinhamento vertical dos itens. É preciso definir que container esteja na direção vertical

align-items: flex-start; align-items: flex-end; align-items: center; align-items: stretch;

flex-wrap

Define se os itens devem quebrar linha. Para ver o efeito, diminua o tamanho da janela, veja o que acontece.

flex-wrap:wrap; flex-wrap:nowrap;

flex-grow, flex-shrink, flex-basis

Controle de tamanho flexível dos itens.

align-content

Controle do espaçamento vertical em múltiplas linhas.
